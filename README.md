# shutter_app

A Flutter camera project.
- Adjust Exposure (EV)
- Choose amount of picture per photo taken
- Crop according to specific width
- Keep user setting data

# SCREEN

- Splash screen
- Main Camera Page
- Image List
- Full Screen Image after crop
